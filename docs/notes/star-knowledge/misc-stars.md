# 三十七杂耀

## 前言

紫微斗数中讲求的是一个 `平衡`。假如过于偏向某一个特性，反而容易深受其害，就比如增加智商的星曜过多，就很容易目空一切，觉得自己才是那个看透世间万物本质的人。这样容易产生什么影响，三国时期的周瑜和诸葛亮的故事已经告诉我们答案了。杂耀除了有些许特点，还能展现出一种气质，平时我们说的某某看起来就一脸苦情相，谁谁有一种忧郁气质等等，就和杂耀还有 [`神煞`](https://docs.iztro.com/learn/dec-star.html) 有关系。

### 天才星

天才星属阴水，智慧才能之星，是具有智慧的星曜，主聪明才艺。故最喜欢与天机星、天梁星、天府星、天相星、天魁星、天钺星、文昌星、文曲星等星曜同宫坐命，同气为大，聪明过人。机月同梁格会天才，智慧很高，辩才特好。

如果天才星没有落在命宫，而是落在福德宫、官禄宫等关键的宫位，组合优良时，也代表此人智慧高超，而且拥有与众不同的思维方式。

为命宫主星时，面青白色，圆长脸，中高身材。聪明外露，有才艺、才干、口才，机警，反应快，正派仁慈，对环境的适应能力很强，能攻能守，能屈能伸。略带神经质，有点傲气。

入身命，逢吉星主聪明机智，才能出众，声誉显扬。其人必有某种天才，格局好者为将相领导的天才;格局差者是小兵的天才。会昌曲魁钺科星，才华出众，有名气。与天机同宫，思想敏捷，擅长策划，有急才。加煞忌，主工于心计，多诡计，常怀才不遇。逢伏兵，有理说不清，事业多阻。遇主星化忌或煞多，小心变成白痴。



### 天厨星

天厨属土，为俸禄之星，主吉。天厨入命、官等宫，主有终身公职;加会廉、贪、巨、阳等星，主多应酬、能研究厨艺;若遇天才星，更显烹饪之资。

天厨入命、官等宫，主有终身公职;加会廉、贪、巨、阳等星，主多应酬、能研究厨艺;若遇天才星，更显烹饪之资。

天厨在命、财、官、迁等宫逢到，若其人命格被冲破，或财、官、迁等宫都不好，这种人一生一事无成，此时天厨代表的是好吃懒做。

命、财、官、迁、福德等宫逢天厨之人，要勤能补拙，不要太重视物欲享受、安逸，否则一生无大做为。

天厨坐命的特性：

1. 福贵(官贵之命主贵，福德宫好则主福)

2. 有口腹之欲。

3. 喜自庖厨。

个性：

1. 随遇而安，懂得享受的乐天派。

2. 口欲很强，对吃有独道的见解。

3. 性格大而化之，有时不爱动脑。

天厨坐命的人，福德宫相当重要，因为他是一个享受的宫位，一个爱吃、想吃的人但他不见得有能力或有口福吃，能不能吃美食来满足他的口腹之欲就要由福德宫来决定了。

看命格：

1. 命格贵，福德宫吉。

　　(1)代表福贵，高官厚爵，有人做给他吃。

　　(2)有口腹之欲。(3)美食主义者。

2. 命格贵，福德宫凶。

　　(1)代表主操心劳力。

　　(2)虽主贵但喜逢之。

　　(3)美食主义。

3. 命格低，福德宫吉。

　　(1)操心劳力。

​		(2)代表喜庖厨，喜欢下厨做给别人吃。

4. 命格低，福德宫凶。

　　(1)主好吃懒做。

　　(2)一生无成。

​        (3)眼高手低。

### 天寿星

天寿星属阳土，长寿星，辅天同天梁星，主增加人的通达和福寿，有化凶解难作用。十二宫中均主福，佐吉而不助凶。须配合得力的主星，最喜同梁二星，独守则力量微弱而无用。

为命宫主星时，面黄白色，圆长或方形脸，中高身材。性忠厚老实，温和谦逊，勤勉，不计较得失。做事认真勤奋，负责，不厌其烦，能力强，智慧高，深谋远虑，安于现状，不重名利，喜助人，不喜嘈杂环境，有孤独感，但感情较理智，见识广。遇空亡则孤独死板，早年不活泼，中年以后经验多，变得相当活跃。老年安享快乐。

在酉戍亥宫为入庙，其它宫位亦不失陷。

老年运遇到天寿之宫，为反格，很可能死亡。入疾厄宫则生命力旺盛。

会机月同梁，老气横秋，主有高寿。但与天梁同守命或疾厄宫时，必有灾病惊险，惟能逢凶化吉，脱险而已。遇羊陀，有损阴德和寿命。

守命身福宫，主享高寿，值庙旺之地尤佳，纵使本宫及三方见煞，亦可抵挡灾祸，惟大小二限及太岁值于凶煞交会之宫，则对灾祸无能为力

天寿星也能够增强紫微星分析、谋划、管理的才能,紫微星与天寿星同宫坐命时，所得到的好处要比与天才星同宫更多。

除了紫微星以外，天梁星、天府星、贪狼星等星也喜与天寿星同宫坐命，可以长寿。
