# 紫微斗数格局



## 前言

在紫微斗数中有非常多的格局，而这些格局就类似于一些化简过后的数学公式。对于解盘而言，即使不知道这些格局，也不影响对星盘的解读。它们就好比游戏里面的一些“套装”，在集齐一些套装以后会有额外的套装属性。然而，千万不能只看格局而忽略了基本的分析，那样是非常容易误入歧途的。
